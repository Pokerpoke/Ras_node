README

1.依赖

  • 编译工具：CMake
  • 日志记录：Log4cpp

  • 文档生成：Doxygen

    • Doxygen依赖Graphviz生成调用关系图

    ```shell sudo apt install graphviz ```

  • 其他依赖：

    • ALSA

    ```shell sudo apt install libasound2-dev ```

    • JRTPLIB

    ```shell git clone https://github.com/j0r1/JRTPLIB.git cd JRTPLIB mkdir build cd build cmake .. make sudo make install ```

    • JRTPLIB依赖JThread

    ```shell git clone https://github.com/j0r1/JThread.git cd JThread mkdir build cd build cmake .. make sudo make install ```

2.编译

  • 编译可执行程序
mkdir build
cd build
cmake ..
make

示例程序位于build/bin/目录下

  • 启用Debug
cmake -DCMAKE_BUILD_TYPE=Debug ..

  • 交叉编译

    • Tiny4412

    修改cmake/toolschain/CMakeLists.txt中的TOOL_CHAIN_DIR为相应地址

    ```shell cmake -DCMAKE_BUILD_TARGET=Tiny4412 .. ```

  • 生成API文档
make doc
  • 在线API参考

Ras_node API
